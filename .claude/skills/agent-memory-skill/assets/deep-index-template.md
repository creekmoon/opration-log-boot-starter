## 文档类型说明

| 类型 | 用途 | 包含内容 |
|------|------|----------|
| **业务流程** (`flow-*.md`) | 描述完整业务执行流程 | 步骤、流程图、数据流转、代码执行链 |
| **生命周期** (`lifecycle-*.md`) | 描述实体状态机 | 状态定义、转换条件、钩子、一致性 |
| **数据流** (`dataflow-*.md`) | 描述数据流转路径 | 来源、转换、存储、输出 |
| **模块交互** (`interaction-*.md`) | 描述跨模块协作 | 调用链、契约、错误处理、一致性 |

## 新增深度文档

如需新增深度文档，请遵循以下命名和目录结构：

```
03-deep/
├── 00-index.md                    # 本文件
├── {模块名}/                       # 对应模块目录
│   ├── flow-{业务流程}.md         # 业务流程
│   ├── lifecycle-{实体名}.md      # 生命周期
│   ├── dataflow-{场景}.md         # 数据流
│   └── interaction-{协作}.md      # 模块交互
└── ...
```
